## Environment

- Node.js
- Electron

## Develop

```sh
# install packages
npm i
# dev serve
npm run serve
# run dev page
npm run dev
# build page
npm run build
# run build page
npm run dev:build
```

# Run

```sh
# install packages
npm i
# build page
npm run build
# run app
npm run start
# run app with pm2
pm2 start npm --name "coin-tray" -- start
```
